Para guardar datos online necesitas conectar Firebase (gratis). Sigue estos pasos una sola vez:
1
Ve a console.firebase.google.com e inicia sesión con Google.
2
Crea un proyecto nuevo → cualquier nombre, desactiva Google Analytics si quieres.
3
Dentro del proyecto: Firestore Database → "Crear base de datos" → modo test → cualquier región.
4
Ve a Configuración del proyecto (⚙️) → pestaña General → baja hasta "Tus apps" → añade una app Web (</>) → copia los datos de firebaseConfig.
